Municipal Office Network Design

Overview
Designed and simulated a secure municipal office network using Cisco Packet Tracer.

Features:
*VLAN Segmentation
*Inter-VLAN Routing
*ACL Security
*DHCP Services
*Public WiFi Network
*Trunking Between Switches
*Redundant Network Links

VLAN Structure

| VLAN | Department       |
| ---- | ---------------- |
| 10   | Mayor            |
| 20   | Treasurer        |
| 30   | Secretary        |
| 40   | RHU              |
| 50   | SK               |
| 60   | Public WiFi      |
| 100  | Municipal Server |

Security
ACLs were implemented to restrict access to the Municipal Server. Only devices within the Mayor VLAN are permitted access.

Technologies Used
-Cisco Packet Tracer
-VLANs
-ACLs
-DHCP
-Static Routing
-Trunking
